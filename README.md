# 🚀 Civic AI

A smart civic issue reporting system built using Flask.

## ✨ Features
- Report issues (pothole, garbage, etc.)
- Auto priority detection
- Admin dashboard
- Real-time stats

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS

## ▶️ Run Locally
pip install flask
python app.py
